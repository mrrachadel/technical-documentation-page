Technical Documentation Page - HTML5

Introduction

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML5 - New Tags

Despite making clear that HTML5 came to make our lives easier, I must say it has brought new tags and new elements which did not exist before.

In this technical documentation page, we will cover just a couple of the tags of HTML5. Such as:

  • Article
  • Header
  • Footer
  • Section

Article

As the name can suggest, the tag < article > define and structure a region of your site in order to make it more independent.

< article > ... < /article >

Header

Before explaining it, do not make a confunsion with < header > e < head >. Like the its name suggests, it is used to build a header, generally about a page's informations such as its name, slogan, brands or titles.

< header > ... < /header >

Footer

The footer are, generally, piece of information of a site that comes at the bottom of each page.

< footer > ... < /footer >

However, contrary to common thinking, it is not mandatory to be in the end of the page.

Its idea is to provide information about the page and its copyrights and list other interesting links.

Section

This tag is used to define and structure many sections in a page. It is very generic and its use is helpful to organize the elements of a page.

< section > ... < /section >

With this HTML5 element, you will be able to group the elements which have something in commun, such as the section in a page of an article, with the author's information and a picture.

Other Elements

The tag list extends. There are many tools and elements to be used in HTML5, such as:

  • Midia elements: video, audio, embed e source;
  • Form elements: input, output, menu, command, progress, datalist, etc;
  • Graphic elements: canvas.
